# Web Core EX

Professional Digital Agency Website built with Next.js 15, React, Tailwind CSS, and Framer Motion.

## Features

- **Next.js 15 App Router**: Server Components for optimal performance and SEO.
- **Responsive Design**: Mobile-first architecture using Tailwind CSS.
- **Animations**: Smooth page transitions and element animations using Framer Motion.
- **Contact Form**: Integrated with EmailJS for real-time inquiries.
- **SEO Optimized**: Dynamic metadata for all pages.

## Tech Stack

- **Framework**: Next.js 15
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion, React Type Animation
- **Icons**: Lucide React, React Icons
- **Deployment**: Netlify / Vercel ready

## Getting Started

1.  **Install dependencies:**
    ```bash
    npm install
    ```

2.  **Run the development server:**
    ```bash
    npm run dev
    ```

3.  **Build for production:**
    ```bash
    npm run build
    ```

4.  **Start production server:**
    ```bash
    npm run start
    ```

## Project Structure

- `src/app`: App Router pages and layouts.
- `src/components`: Reusable UI components organized by feature (home, about, etc.).
- `public`: Static assets (images, icons).
